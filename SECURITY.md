# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| v0.1.x  | ✅ Active |

## Reporting a Vulnerability

Open an issue at
[github.com/wildfirebill-unraid/mcp-gateway-unraid/issues](https://github.com/wildfirebill-unraid/mcp-gateway-unraid/issues)

Do **not** expose the gateway port (8811) to the public internet. This
gateway is designed for LAN-only use behind your firewall. Set a strong
`MCP_GATEWAY_AUTH_TOKEN` and keep your Unraid system updated.
